# Gamma Slides Website

An AI-powered platform for generating professional websites in both **Offline** and **Online** modes using Gamma. The platform enables users to create modern, responsive, and visually appealing websites with intelligent content generation, customizable templates, and seamless deployment workflows.

---

## 🚀 Features

- 🌐 Website generation in **Offline** and **Online** modes
- 🤖 AI-powered website creation using Gamma
- 📱 Fully responsive and mobile-friendly designs
- 🎨 Modern UI with customizable layouts and themes
- ⚡ Fast website generation with optimized performance
- 📄 Automatic content and section generation
- 🖼️ Image and media integration
- 🔍 SEO-friendly page structure
- 📂 Export generated websites for local hosting
- ☁️ Deploy websites directly to online hosting platforms
- 🔄 Easy project management and regeneration
- 🔒 Secure and scalable architecture

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/SiddharthS-Dev/Gamma-slides-Website.git
cd Gamma-slides-Website
```

### Frontend

```bash
cd slidevault/frontend
npm install
npm run dev
```

### Backend

```bash
cd slidevault/backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
